# Incident Reports

Post-incident analysis reports documenting attack simulations run in this SOC lab.

Each report follows the [TEMPLATE.md](TEMPLATE.md) format and includes:
- Executive summary
- Chronological timeline (from Kibana)
- MITRE ATT&CK kill chain mapping
- Indicators of Compromise (file hashes, IPs, registry keys)
- Detection details (which rules fired, latency, gaps)
- Remediation recommendations
