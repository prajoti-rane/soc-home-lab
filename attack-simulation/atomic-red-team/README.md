# Atomic Red Team — Technique Selection and Execution

Atomic Red Team (ART) by Red Canary provides small, focused tests for individual MITRE ATT&CK techniques. This lab uses ART to validate Wazuh detection coverage.

## Installation

ART is run from the victim-windows VM:

```powershell
# [windows] Install Invoke-AtomicRedTeam PowerShell module
Install-Module -Name invoke-atomicredteam -Scope CurrentUser -Force
Import-Module invoke-atomicredteam

# Download atomics (technique definitions)
Invoke-AtomicTest T1059.001 -GetPrereqs
```

## Techniques Used in This Lab

| ID | Technique | Test # | Prereqs | Notes |
|----|-----------|--------|---------|-------|
| T1059.001 | PowerShell | 1 | None | Encoded command execution |
| T1547.001 | Registry Run Key | 1 | None | HKCU persistence |
| T1003.001 | LSASS Memory | 1 | Admin | Requires elevated prompt |
| T1070.001 | Event Log Clear | 1 | Admin | Clears Security log |
| T1055 | Process Injection | 1 | Admin | Allocates shellcode |
| T1046 | Network Scan | — | Nmap on Kali | Use `nmap` from kali instead |
| T1074.001 | Local Data Staging | 1 | None | Copies files to %TEMP% |

## Running Tests

```powershell
# [windows] Run a single test
Invoke-AtomicTest T1059.001 -TestNumbers 1

# Show what a test will do without executing
Invoke-AtomicTest T1059.001 -TestNumbers 1 -ShowDetails

# Check prerequisites
Invoke-AtomicTest T1003.001 -TestNumbers 1 -CheckPrereqs

# Clean up after a test
Invoke-AtomicTest T1547.001 -TestNumbers 1 -Cleanup
```

## Expected Alert Latency

| Stage | Typical latency |
|-------|---------------|
| Sysmon event generated | < 1 second |
| Wazuh agent forwards to manager | 1–5 seconds |
| Wazuh manager processes + indexes | 2–10 seconds |
| Alert visible in Kibana | 5–15 seconds total |

## Tip: Correlate ART Run Time with Kibana

Before running ART tests, note the exact time in UTC:

```powershell
[System.DateTime]::UtcNow.ToString("yyyy-MM-ddTHH:mm:ssZ")
```

Use this timestamp as the "from" filter in Kibana to find alerts generated by that specific test run.
